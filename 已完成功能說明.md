### 2. 動畫效果系統 ✨

#### 🎬 全新動畫庫
為您的系統加入專業級動畫效果，大幅提升用戶體驗！

**新增檔案：**
1. **`css/animations.css`** - 完整動畫樣式庫
2. **`js/animation-utils.js`** - 動畫工具函數

#### 🌟 動畫效果清單

##### 頁面動畫
- 頁面淡入過渡
- 平滑滾動
- 元素滑入效果

##### 卡片動畫
- 依序出現動畫（stagger effect）
- Hover 提升效果
- 點擊反饋

##### 按鈕動畫
- **漣漪效果** - 點擊時的水波紋擴散
- **彈跳效果** - 輕微的彈性反饋
- **脈衝效果** - 吸引注意力的脈衝動畫

##### 載入動畫
- 旋轉載入器
- 脈衝載入
- 點點載入
- 骨架屏閃爍

##### 特殊效果
- **數字滾動** - 數字從 0 漸變到目標值
- **Toast 通知** - 滑入滑出動畫
- **Modal 彈窗** - 優雅的彈出效果
- **徽章彈出** - 突出顯示新內容
- **圖示動畫** - 搖晃、心跳、旋轉等

#### 🛠️ 使用方式

**自動應用（已整合）：**
系統會自動為以下元素添加動畫：
- 所有卡片依序出現
- 按鈕漣漪效果
- 列表項目滑入
- 數字滾動計數

**手動使用：**
```html
<!-- 淡入效果 -->
<div class="animate-fade-in">內容</div>

<!-- 滑入效果 -->
<div class="animate-slide-up">內容</div>

<!-- 卡片 hover 提升 -->
<div class="card-lift">卡片內容</div>

<!-- 按鈕脈衝 -->
<button class="btn-pulse">重要按鈕</button>

<!-- 數字滾動 -->
<div data-count-up>1000</div>
```

**JavaScript API：**
```javascript
// 數字滾動動畫
AnimationUtils.countUp(element, 1000, 1500, 'NT$ ', '');

// 元素淡入
AnimationUtils.fadeIn('.my-element', 200);

// 卡片依序動畫
AnimationUtils.staggerCards('.my-container');

// 平滑滾動
AnimationUtils.scrollTo('#target-element');
```
